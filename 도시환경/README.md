## EDA 프로젝트 요약

1. 프로젝트 주제 및 목적
        - 주제 및 목적에 대해 간단히 설명 

2. 데이터 전처리

        - 데이터 별 전처리 과정을 간단히 설명, 관련 코드가 있다면 [링크](깃허브 내 파일 주소)를 걸어주세요
            
 
3. 분석 방법 및 결과
    
        - 분석 방법 및 결과를 설명, 관련 코드가 있다면 [링크](깃허브 내 파일 주소)를 걸어주세요
		    
4. 결론

        - 최종 결론을 적어주세요
    
5. 아쉬운 점
    
        - 한계점을 적어주세요

6. 추가로 하면 좋을 분석 방법
    
        - 생각해보신 발전 방향이 있을 경우 적어주세요
